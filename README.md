# Village Defence 保卫村庄

## Quick Start

### Windows

#### MinGW + Ninja

```cmd
debug-ninja.bat
```

#### Visual Studio

```cmd
release-vs.bat # or debug-vs.bat
```

### Linux

#### Ubuntu:latest

```bash
sudo bash ./debug.sh
```
