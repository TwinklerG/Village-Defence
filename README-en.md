## Quick Start

Recommended OS: Windows/Linux

Recommended CMake generator: Visual Studio/Ninja

### Windows

**MinGW + Ninja**

```cmd
debug-ninja.bat
```

**Visual Studio**

```cmd
release-vs.bat # or debug-vs.bat
```

### Linux

**Ubuntu:latest**

```bash
sudo bash ./debug-ubuntu.sh
```
